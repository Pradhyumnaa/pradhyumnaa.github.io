BookNest

BookNest is an app that allows users to put up Paperback books and E-Books they own for other users to borrow for a definitive period of time. It also allows users to exchange books with other users. There will be a map which showcases what books other users have put up for exchange and it allows the user to schedule a meeting with them to exchange books. There will also be a wishlist function, which allows the user to add a book they want to that list. Once that book is available in the collection of books available for lending, the user gets a notification.
